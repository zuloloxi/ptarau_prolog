Prolog Server Chat Agent Output Window

If your browser and OS supports them, Microsoft Agent components
will be loaded and used for speech synthesis. Please also make sure
that your windows\msagent\chars directory contains the agents Merlin
and Peedy. Windows XP users will need to download and install the
SAPI 4.0a runtime and TTS engine. After choosing a login name and
a password, the agent will remember you and display your past 
interaction in a history window. Note that our chat agent is is not
based on handcoded rules; instead it uses a generic inference engine
that extracts answers from the WordNet and OpenMind knowledge bases.

Have fun chatting with the agent, but keep in mind that:

A conversational agent is an approximation of a blind, paralized
from the neck down person, who also happens to be deprived of a few 
other sensorial functions, while being held in solitary confinment :-)
